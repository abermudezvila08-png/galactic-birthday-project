# Arquitectura del Proyecto

El Galactic Birthday Project está organizado como una producción cinematográfica.

## 01. Documentos

Contiene toda la documentación del proyecto.

Ejemplos:

- Manifiesto
- Hoja de ruta
- Principios narrativos
- Arquitectura

## 02. Investigación

Toda afirmación científica debe poder rastrearse hasta una fuente.

## 03. Guion

Escenas completas del documental.

Una escena por archivo.

## 04. Storyboard

Describe visualmente cada escena antes de producirla.

## 05. Recursos

Imágenes, sonidos, música, logotipos y animaciones.

## 06. Versiones

Cada versión importante del proyecto tendrá su propio registro de cambios.

El proyecto debe poder crecer durante años sin perder el orden.
