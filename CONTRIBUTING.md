# Contributing to Galactic Birthday Project

Welcome.

This project combines science, storytelling, philosophy, art and technology to tell one story:

> The Earth is only twenty.

We welcome contributions in:

- Astronomy
- Geology
- Biology
- History
- Philosophy
- Theology (clearly identified as philosophical or theological reflection)
- Cinematic storytelling
- Illustration
- Music
- Translation

## Our Rules

1. Respect scientific evidence.
2. Respect different worldviews.
3. Distinguish facts from narrative interpretation.
4. Improve before replacing.
5. Leave the project better than you found it.

Every contribution should help future generations understand our place in the Universe.
